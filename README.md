## 这是一个学习demo

TypeScript + mocha 学习demo.


**项目的目录解构如下所示**

```
mocha-ts-demo
    |—— dist                 // 存放编译后的代码
    |—— mochawesome-report   // 生成的 mocha html报告
    |—— node_modules         // 项目依赖
    |—— src                  // ts源码
    |—— test                 // 测试脚本
    |—— tsconfig.json        // ts 配置文件
```
###
```
yarn test 执行测试脚本
yarn dev  执行src/index.ts文件
```
